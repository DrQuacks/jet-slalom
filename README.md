# Jet Slalom TS

Small TypeScript canvas homage to the classic Flash-style Jet Slalom game.

## Run

1. From `/Users/kellar/Documents/Playground`, run `npm run start`
2. Open `http://localhost:4173`

## Notes

- The TypeScript source lives in `src/main.ts`
- A browser-ready `dist/main.js` is included so the game runs immediately without installing packages
- If you install dependencies later, `tsconfig.json` is already in place for compiling the TS source
